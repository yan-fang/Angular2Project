# Ease Web Version 2

[![Commitizen friendly](https://img.shields.io/badge/commitizen-friendly-brightgreen.svg)](http://commitizen.github.io/cz-cli/)

## Prerequisites
 - node version >= 6.4.9
 - yarn installed

## Start the project
  - Run `yarn`
  - Run `yarn run dev`
  - Browse to http://localhost:3000

## Commit Message Guidelines
  - Run `yarn run commit`
